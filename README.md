# Animated Login Page 🐻

A delightful and interactive login page built with React, featuring smooth animations and a playful design. The page includes animated elements that respond to user interactions, creating an engaging login experience.

![preview](./public/demo.gif)

## ✨ Features

- Interactive animations on form interactions
- Responsive design that works on desktop and mobile
- Modern and playful UI/UX
- Form validation with animated feedback
- Dark mode support (if implemented)

## 🛠️ Tech Stack

- **React 19** - UI Library
- **TypeScript** - Type safety and better developer experience
- **Vite** - Next Generation Frontend Tooling
- **Tailwind CSS** - Utility-first CSS framework
- **React Compiler** - For optimized performance

## 🚀 Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm, yarn, or pnpm

### Installation

1. Clone the repository
```bash
git clone https://github.com/dannybarajas/animated-login.git
cd animated-login
```

2. Install dependencies
```bash
npm install
#yarn
#pnpm install
```
3. Start the development server
```bash
npm run dev
#yarn dev
#pnpm dev
```


4. Open your browser and visit `http://localhost:5173`

## 📝 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint for code checking

## 🎨 Customization

You can customize the animations and styling by:
- Modifying the Tailwind configuration in `tailwind.config.js`
- Adjusting animation parameters in the component files
- Updating color schemes and themes
